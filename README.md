# img2ascii

Image processing to ascii art made as one of my first Rust projects.
Any image that that the [Image Crate](https://crates.io/crates/image) can parse for RGB data can be made into ASCII art.
There is support for image scaling by defining what percent of the pixels are converted to ASCII characters.
